findcell(q, V)

Use bisection search to find the cell containing q, assuming V is a sorted vector of coordinates. The returned integer is the index of the element in V immediately less than q. For example, if findcell returns 2, then q âˆˆ [V[2],V[3]). If q is less than every element in V, 1 is returned, indicating the first cell in V.
